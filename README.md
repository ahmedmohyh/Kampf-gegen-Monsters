# Kampf-gegen-Monsters
Es geht in diesem Spiel darum dass, ein Spiel im Kampf gegen Monsters eingestzt wird. Das Ziel des Spiels ist dass der Spiler maximum überlebend zu halten. 
Der Spieler kann gleichzeitig mehr als ein Monster kämpfen, durch dass er ein sogenannte "Special Attack" oder ein "Sword Attack" durchführt.
